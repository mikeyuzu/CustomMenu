メニューの項目で「合成」を押したときの挙動を実装します。
ここはhttpアクセスではなく固定のメニューを出します。
リストは次の3つ
合成倉庫
アイテム別リスト
ギルド別リスト

また、このリストを表示したときにメニューの右側にサブウィドウを表示し、説明文を入れます。
合成倉庫の説明文
```
合成倉庫はモンスターやショップから合成素材を入手した時に自動的に格納される倉庫です\n格納した素材が合成に可能な数になった時、合成レシピが開放されます\n開放された合成レシピから合成が可能になります\n\n合成に失敗はありません\nHQ合成にはその合成アイテムに必要なスキル+10が必要になります\nHQ2には+20、HQ3には+30が必要になります\n\n合成したアイテムはポストに届きます\n\n合成倉庫からマイバックに取り出したい時は、その項目を選んで取り出してください\n取り出したアイテムはポストに届きます\n\n取り出した素材を再び合成倉庫に移動させたい場合は\nポスト以外からマイバックへ移動する操作で自動判別されて合成倉庫に移動します\n例えばモグケースからマイバックへ移動など
```

アイテム別リストの説明文
```
アイテムの種別（武器、防具など）からレシピを検索します。
```

ギルド別リストの説明文
```
ギルドを選んでレシピを検索します。
```
